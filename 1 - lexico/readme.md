# Usage

## Scanner

```bash
python3 bminor.py --scan test/syntax/good01.bminor
```

## Parser

```bash
python3 bminor.py --parse test/syntax/good01.bminor
```

## AST

```bash
python3 bminor.py --ast test/syntax/good01.bminor
```

## Checker

```bash
python3 bminor.py --checker test/syntax/good01.bminor
```
